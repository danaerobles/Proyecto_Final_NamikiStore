# NamikiRoute - Starter kit
Proyecto original para una app de planificación y optimización de rutas.

Estructura:
- backend/ (Node + Express + solver runner)
- frontend/ (Vite + React + Leaflet)
- samples/ (ejemplos CSV)
- docker-compose.yml, Dockerfiles

Sigue las instrucciones en backend/ y frontend/ para correr localmente.
